# BestSudokuCalculator

L'idea del progetto è nata tra me e un altro ragazzo e con l'aiuto di un altro paio di persone abbiamo creato questo piccolo algoritmo che risolve i sudoku(non quelli della sfida impossibile della settimana sudoku)

Ad oggi, l'algoritmo risolve i sudoku di livello facile e intemedio. Quelli più complessi sono ancora in fare di sviluppo. 

All'interno del file Sudoku.java sono presenti tre esempi di sudoku ordinati dal più facile al più complesso. I primi due sudoku verranno eseguiti senza troppi problemi mentre il terzo creerà un loop infinito.
